The RAID function in {PLATFORM} combines the /home partition and one external hard disk into a single logical unit.

Important

  • Warning! The process will destroy all user data - both on the internal and external disk.
  • An old RAID disk will automatically be connected and included into the {PLATFORM} system upon boot, even though you have not set up your {PLATFORM} for a RAID extension. It is impossible to remove the extension without reinstallation. To format an old external RAID extended disk, connect the disk to a already running {PLATFORM}. Then choose 'Format' in the 'Disk -> Information' menu.
  • To get out the most of your disks, use an external disk in the same size as your internal disk in {PLATFORM}.
  • Total capacity of the array equals the capacity of the smallest disk in the array.
  • It will take some time to create or restore a RAID array. For a 1TB disk it will take about 4 hours. This is handheld in the background of the {PLATFORM} system and will not be indicated to the user.
  • To reuse your {PLATFORM} standalone without the RAID setup you have to reinstall your {PLATFORM} system.

Create RAID array

  1. Attach an eSATA disk. Please notice that the disk should be pre formatted, without old RAID or LVM systems.
  2. Click 'Create RAID array'
  3. Select which external disk to include in the array.
  4. Warning! All data on both disks will be erased. Click 'Create RAID'.
  5. Wait for the progress bar to complete. Be patient, it will take some time to create a RAID array if you are using large disks.
  6. When finished, your external disk has been included in your RAID array.

RAID status

List of RAID arrays - shows the total disk space, i.e. the smallest available disk (the {PLATFORM} /home partition or the external eSATA disk) in the array.

List of RAID disks - shows the disks attached to the RAID system.

Recover an external disk

If you have a disc failure on your external disk or if you have by mistake disconnected the external disk you will need to do the following.

  1. Disconnect the broken external disk from the backside of {PLATFORM}.
  2. Attach a new external disk (or reconnect the by mistake disconnected one).
  3. Click on 'Recover RAID array'.
  4. Select which external disk to add to RAID array. Click 'Recover array'.
  5. Warning! All data will be erased from the external disk. Click 'Recover array'.
  6. Press 'Close' to continue working with {PLATFORM}. The synchronization is in progress and will take a while.

Recover an internal disk

If you have a disk failure on your internal disk in {PLATFORM} you have to start by exchanging the disk and reinstalling via an USB memory. Then update {PLATFORM} to the latest software.

  1. Attach the external RAID disk that is containing your data.
  2. Click on 'Recover RAID array'.
  3. Select the external disk to add to RAID array. Click 'Recover array'.
  4. Warning! All user data and the storage area will be erased from the internal disk. Click 'Recover array'.
  5. Press 'Close' to continue working with {PLATFORM}. The synchronization is in progress and will take a while.
