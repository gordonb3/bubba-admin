This backup service allows you to backup any data on your {PLATFORM} to a USB disk, an eSATA disk or to a remote server supporting FTP or SSH protocols. For instance you can create a backup from one {PLATFORM} to another using the SSH protocol, and using the EasyFind name as the “target server”.

The backup service is timelined, hence you can choose to restore your data from any of the backup occasions. For instance if you create a backup job you run hourly, and need to restore a file to the state it had four hours earlier, this is possible.

The backup can also be run differentially. This means that it will only send changes that might have taken place in your data. If no changes are made, no data is being sent and no additional storage space is used on the target.

You can also choose to encrypt the backup. If you choose this, the backup is encrypted both during transfer to target and on the actual target. This allows you to send backups to untrusted servers without worrying that someone might read your backup data.

Note: If you misplace the password you use to create an encrypted backup, you will not be able to restore this backup!

Create a backup

Click the button “Add new backup job” and follow the wizard

Restore a backup

Click the button “Restore” next to the date or time you wish to restore from.

Restore a backup not using {PLATFORM}

If you in the future would like to restore a backup not using {PLATFORM}, this is possible but does require a Linux machine with the Duplicity software installed. Just point Duplicity to the backup target and perform the restore. Note that you still ned the password if you chose to encrypt your backup

How long will my backup take?

The first time a backup runs it can be very time consuming. For example 500 digital photos of 5 MB each would take approximately five hours to complete the first time, over a fast internet connection or to a local device. Note: The following backups only sends changes (new photos) so they will be faster.

Save your backup jobs

As a security measure, perform a settings backup to save your backup jobs if you have to reinstall your {PLATFORM}. Click Settings in the menu and then Backup settings
