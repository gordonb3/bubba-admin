From the File manager you are able to reach your files on {PLATFORM} even if you are away from home.

Navigation

Navigate into a folder by clicking the arrow to the right of the folder name or double clicking it, navigate up one level by clicking the left arrow or click the /home/username/ line.

File and folder actions

Click files or folders and use the icons in the file function bar to perform different actions. The functions from left to right are:

  • Create folder
  • Upload file
  • Download as zip
  • Move files
  • Copy files
  • Rename
  • Change permissions
  • Add to album
  • Delete

Photo album

Logged in as administrator you are not able to add photos to the {PLATFORM} photo album. Log in with your standard user to add and manage photos and your albums.
