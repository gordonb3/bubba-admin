Software update

{PLATFORM}'s software can easily be updated to gain new functionality. Press 'Update system', and the update is automatically performed. Please have patient, it might take a while to perform an update.

After an update, status information is shown. Press the '+' sign to show the complete message.

Hotfixes

Before an update, the software will communicate and exchange information to an Excito controlled server for an automatied analyzis.

This system, which is called "hotfixes", will send information regarding your {PLATFORM} for an automatic analysis, and based upon this information, appropriate actions can be issued to your {PLATFORM} to be taken before the main upgrade is initiated.

All this information is sent to Excito via an encrypted connection, and all server responses are signed using GPG to ensure the security that the content is distributed by Excito.

This feature cannot be disabled from the user interface. If you wish to opt out, We refer to the user manual how to proceed with an manual upgrade from the command line interface.

Any information collected by Excito will never be exposed to any other party than Excito. The information will only be used to pin point errors and perform the best matched updates for your {PLATFORM}.

All information that is collected is listed below.

  • MAC address
  • IP addresses
  • Serial number and key
  • RAM memory size
  • CPU model
  • Status for the installed {PLATFORM} specific packages
  • The log file /tmp/{PLATFORM}-apt.log if existing
  • Current software release version
  • Current running kernel
  • Root disc partition usage
  • Disc partition setup (LVM/RAID)
