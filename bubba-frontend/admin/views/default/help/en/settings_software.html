Software update

{PLATFORM}'s software can easily be updated to gain new functionality. Press 'Update system', and the update is automatically performed. Please have patient, it might take a while to perform an update.

After an update, status information is shown. Press the '+' sign to show the complete message.

Hotfixes

This feature collects status information from your {PLATFORM} and sends to Excito for automated analysis. The update server can then take appropriate actions if your system has problems. The system information is sent to Excito via an encrypted connection, and server responses are GPG signed to ensure the security and that the content is distributed by Excito.

By default this feature is enabled. Use the switch to disable Hotfixes. If doing so you might not get the most important updates to your {PLATFORM}.

Any information collected by Excito will never be exposed to any other party than Excito. The information will only be used to pin point errors and perform the best matched updates for your {PLATFORM}.

All information that is collected is listed below.

  • MAC address
  • IP addresses
  • Serial number and key
  • RAM memory size
  • CPU model
  • Status for the installed {PLATFORM} specific packages
  • The log file /tmp/{PLATFORM}-apt.log if existing
  • Current software release version
  • Current running kernel
  • Root disc partition usage
  • Disc partition setup (LVM/RAID)
